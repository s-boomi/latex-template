
# LaTeX Workflow Template

# Prerequisites

You need to have $\LaTeX$ installed on your computer:

* TexLive: [Installation instructions](https://www.tug.org/texlive/quickinstall.html)
* TintTex: [Website](https://yihui.org/tinytex/)
* MiKTeX: [Website](https://miktex.org/) (Needs Perl to work)

For more help, you can customize LaTeX recipes by following [LaTeX-Workshop's recipes](https://github.com/James-Yu/LaTeX-Workshop/wiki/Compile#latex-recipes).

# Usage

This template can be cloned or forked for your convenience to create $\LaTeX$ documents for the most classic cases, without needing to create an Overleaf account.

Once your $\LaTeX$ installation is set, VSCode users can install the recommended extensions, as [Latex-Workshop](https://marketplace.visualstudio.com/items?itemName=James-Yu.latex-workshop) is essential to keep your workflow comfortable.
