
# LaTeX Workflow Template

You need to have LaTeX installed on your computer:

* TexLive: [Installation instructions](https://www.tug.org/texlive/quickinstall.html)
* TintTex: [Website](https://yihui.org/tinytex/)
* MiKTeX: [Website](https://miktex.org/) (Needs Perl to work)

For more help, you can customize LaTeX recipes by following [LaTeX-Workshop's recipes](https://github.com/James-Yu/LaTeX-Workshop/wiki/Compile#latex-recipes).
